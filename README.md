# Функціональність програми
- створення об’єктів класу GameCharacter;
- збереження об’єктів у колекції `List<GameCharacter>`;
- перегляд усіх створених об’єктів у табличному вигляді;
- пошук об’єкта за характеристикою;
- видалення об’єктів;
- демонстрація поведінки персонажа (атака, отримання шкоди).

# Реалізація принципів ООП
У програмі реалізовано:
- інкапсуляцію шляхом використання `private`-полів;
- доступ до полів виключно через `public`-властивості;
- перевірку коректності значень у секціях `set`;
- обробку помилок за допомогою механізму `exception`;
- автоматичну властивість;
- обчислювальну властивість;
- властивість з різним рівнем доступу для `get` та `set`;
- використання `private`-методів у `public`-методах.

# Структура проєкту
- `Program.cs` — основна логіка програми та меню;
- `GameCharacter.cs` — реалізація класу з інкапсуляцією та властивостями;
- `CharacterClass.cs` — перелік можливих класів персонажа (enum).

# Як запустити програму
1. Клонувати репозиторій або відкрити проєкт у Visual Studio.
2. Запустити консольний застосунок (Start або Ctrl + F5).
3. Користуватися меню програми відповідно до інструкцій у консолі.
